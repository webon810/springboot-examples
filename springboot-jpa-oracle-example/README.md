# springboot-example
springboot-jpa-oracle-example

## What you'll need
JDK 8+ or OpenJDK 8+

Maven 3+

Oracle 12c


```
$ mvn install:install-file -Dfile=C:\app\oracle\12.2.0\dbhome_1\jdbc\lib\ojdbc8.jar -DgroupId=com.oracle 
	-DartifactId=ojdbc7 -Dversion=12.2.0.1 -Dpackaging=jar
```
